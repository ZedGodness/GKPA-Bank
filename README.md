# GKPA Bank – Inter-bank Transaction System

A **Full-stack** project simulating a real-world banking ecosystem for inter-bank transactions. **GKPA Bank** features a robust backend infrastructure, complex transaction logic via a central registry, and a modern user interface inspired by leading fintech standards.

## Key Features
* **Transfer Initiation:** Supports both intra-bank (GKPA to GKPA) and inter-bank transactions.
* **Central Bank Registry:** Dynamic routing of payments to the correct bank APIs within the network.
* **Validation & Security:** Real-time IBAN structure verification, balance checks, and payment limit enforcement.
* **Transaction History:** Searchable and detailed visualization of payment history.
* **Modern GUI:** Intuitive Dashboard featuring an adaptive design and professional banking aesthetics.

## Tech Stack
* **Frontend:** React (Vite), Modern CSS (Flexbox/Grid).
* **Backend:** Node.js, Express.
* **Database:** MySQL (mysql2 with Connection Pooling).
* **Protocols:** RESTful API, JSON communication.

## Installation
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the backend: `node server.js`.
4. Start the frontend: `npm run dev`.
